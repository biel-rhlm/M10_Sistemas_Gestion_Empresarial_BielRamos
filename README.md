# Lista de Tareas hecho por Biel Ramos

Este módulo de Odoo permite gestionar tareas internas de manera sencilla.  
Funciones principales:

- Crear tareas con nombre y prioridad.
- Marcar tareas como realizadas.
- Calcular automáticamente si una tarea es urgente (prioridad > 10).
- Visualizar las tareas en una vista de lista y formulario.

Módulo pensado para aprender la estructura básica de Odoo y cómo crear un módulo funcional.
